🚗 AutoMob Mechanic

A responsive multi-page automobile service website developed using HTML5 and CSS3.

📌 Project Overview

AutoMob Mechanic is a car service booking website that allows users to:

- Explore available automobile services
- View service details and pricing
- Book vehicle maintenance services
- Receive booking confirmation

The project is designed with a mobile-first responsive approach and adapts seamlessly across desktop, tablet, and mobile devices.

✨ Features

- Responsive web design
- Multi-page website
- Service listing page
- Service details page
- Vehicle booking form
- Booking confirmation page
- Mobile-friendly navigation
- Flexbox-based layouts
- CSS Media Queries for responsiveness
- GitHub Pages deployment

🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- GitHub Pages
